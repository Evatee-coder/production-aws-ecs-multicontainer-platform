# production-aws-ecs-multicontainer-platform
A production-grade multi-container application platform built on AWS ECS Fargate using advanced Terraform and GitHub Actions CI/CD. Deploys a Flask, Nginx, Celery and Redis stack backed by PostgreSQL on RDS Aurora, with environment-specific infrastructure for development and production. Demonstrates advanced Terraform patterns including for_each loops, map transformation, template files and conditional expressions, ECS service connect for inter-container DNS resolution, AWS Secrets Manager integration, and automated zero-downtime rolling deployments; mirroring real enterprise DevOps engineering standards.
